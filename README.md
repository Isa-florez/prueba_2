instructions to clone
Go to the repository on Github, click on the code button and copy the URL; open the terminal and go to the folder where you are going to save the project, run the command Git clone and paste the link, press enter and then go into the folder and open the file.

With the Live Server extension in Visual Studio, you go to the index page and select the option to open with Live Server, and the page will open.
